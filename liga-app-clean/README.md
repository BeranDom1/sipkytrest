# Black–Yellow responsive theme (v2025-08-22)

Co přináší:
- moderní **černo–žlutý** vzhled,
- tmavý horní pruh s logem a výběrem ročníku,
- levý panel jako **sidebar** (na mobilu se vysouvá tlačítkem),
- čisté tabulky s černo–žlutou hlavičkou a zvýrazněním 1.–3. místa,
- zachovaný sticky footer + scroll jen uvnitř tabulek.

## Instalace
1) Nakopírujte soubory do kořene `liga-app/` a **přepište** `header.php`, `footer.php`.
2) Přidejte složku `assets/` s `theme.css` a `theme.js` (součást tohoto balíčku).
3) V `header.php` případně upravte `$BASE_URL = '/liga-app'` podle skutečné cesty.
4) Proveďte **hard reload** (Ctrl+F5).

V případě potřeby doladím barvy/typografii přesně podle vašich preferencí.
